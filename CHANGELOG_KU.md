### 1️⃣ Widget بۆ شاشەی سەرەکی 📱

**چی زیادکراوە؟**
```
✅ حیسابکەرێکی خێرا لەسەر شاشەی سەرەکی مۆبایل
✅ پیشاندانی دوایین هەژمارکردنی تۆمارکراو
✅ پیشاندانی کۆی تێچوو بە دینار
✅ پیشاندانی بڕی کیلۆوات
✅ کلیک لەسەری بۆ کردنەوەی ئەپ
```

**چۆن کاردەکات؟**
```
کاتێک داتایەک تۆمار دەکەیت:
 ↓
Widget بە ئۆتۆماتیک نوێدەبێتەوە
 ↓
دوایین هەژمارکردنەکەت پیشان دەدات
```

**دیزاین:**
```
┌─────────────────────┐
│       ⚡            │  ← ئایکۆنی کارەبا
│  هەژمارکردنی کارەبا  │  ← ناونیشان
│                     │
│     1500 kWh       │  ← بڕی کارەبا (سپی)
│   205,000 دینار    │  ← تێچوو (سەوز)
└─────────────────────┘
```

**پشتیوانی:**
- ✅ Android (هەموو وەشانەکان)
- ✅ iOS (هەموو وەشانەکان)

### 2️⃣ هەڵگرتن و گەڕاندنەوەی داتا 💾

**تایبەتمەندییەکان:**

#### هەڵگرتن (Save):
```
✅ تۆمارکردنی هەموو هەژمارکردنەکان
✅ ناونیشانێکی دڵخواز بۆ هەر هەژمارکردنێک
✅ تۆماری بەروار و کاتی هەڵگرتن
✅ تۆماری جۆری هەژمارکردن (ماڵان، بازرگانی، هتد)
✅ تۆماری بڕی کارەبا
✅ تۆماری کۆی تێچوو
```

#### گەڕاندنەوە (Load):
```
✅ دەستگەیشتن بە هەموو هەژمارکردنە هەڵگیراوەکان
✅ بینینی وردەکاریی تەواو
✅ گەڕاندنەوەی هەر هەژمارکردنێک بە کلیکێک
✅ زانیاریەکە دێتە ناو فیڵدەکانەوە
✅ ئامادەیە بۆ دەستکاری یان بینینی ئەنجام
```

#### سڕینەوە (Delete):
```
✅ سڕینەوەی هەر هەژمارکردنێک
✅ پرسینی دڵنیایی پێش سڕینەوە
✅ نەمانی گەڕانەوە دوای سڕینەوە
```

**دیزاینی لیستی هەڵگیراوەکان:**
```
┌─────────────────────────────┐
│  📑 ماڵەکەم                │
│  1200 kWh → 95,000 دینار   │
│  هەڵگیراوە لە 5/2/2026      │
│                   ⬇️  🗑️   │
├─────────────────────────────┤
│  🏢 دووکانەکەم             │
│  2500 kWh → 462,500 دینار  │
│  هەڵگیراوە لە 3/2/2026      │
│                   ⬇️  🗑️   │
└─────────────────────────────┘
```

**دوگمەکان:**
