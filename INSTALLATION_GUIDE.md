## 📱 تایبەتمەندییە نوێیەکان | New Features

### 1️⃣ Widget بۆ شاشەی سەرەکی | Home Screen Widget

#### چۆن کاردەکات؟
کاتێک هەژمارکردنێک تۆمار دەکەیت، Widget بە ئۆتۆماتیک نوێدەبێتەوە!

#### زیادکردنی Widget:

**Android:**
1. گوێزراوە لەسەر شاشەی سەرەکی پەنجە درێژ بدە
2. `Widgets` یان `ویجێت` هەڵبژێرە
3. `هەژمارکردنی کارەبا` بگەڕێ
4. Widget ڕابکێشە بۆ شاشەی سەرەکی

**iOS:**
1. گوێزراوە لەسەر شاشەی سەرەکی پەنجە درێژ بدە  
2. دوگمەی `+` کلیک بکە
3. `هەژمارکردنی کارەبا` بگەڕێ
4. Widget هەڵبژێرە و زیادی بکە

---

### 2️⃣ هەڵگرتن و گەڕاندنەوەی داتا | Save & Load Data

#### هەڵگرتنی هەژمارکردن:
```
1. نرخ هەژمار بکە
2. کلیک لە "هەڵگرتنی زانیاری" بکە (دوگمەی سەوز)
3. ناوێک داخڵ بکە (نموونە: "ماڵەکەم")
4. کلیک لە "تۆمارکردن" بکە
5. ✅ تۆمار کرا!
```

#### گەڕاندنەوەی داتا:
```
1. کلیک لە "زانیاری هەڵگیراوەکان" بکە (دوگمەی شین)
2. لیستی هەموو هەژمارکردنەکانت دەبینیت
3. کلیک لە ⬇️ (download icon) بکە
4. داتاکە دێتە ناو فیڵدەکانەوە
5. ✅ گەڕێندرایەوە!
```

#### سڕینەوەی داتا:
```
1. کلیک لە "زانیاری هەڵگیراوەکان" بکە
2. کلیک لە 🗑️ (delete icon) بکە بۆ ئەو هەژمارکردنەی دەتەوێت بیسڕیتەوە
3. "بەڵێ" هەڵبژێرە
4. ✅ سڕایەوە!
```

## 🐛 چارەسەری کێشەکان | Troubleshooting

### کێشە: Widget پیشان نادرێت

**چارەسەر:**
```bash
1. flutter clean
2. flutter pub get
3. flutter run
4. Widget دووبارە زیاد بکە
```

### کێشە: داتاکان تۆمار نابن

**چارەسەر:**
```
1. دڵنیابە کە ناوێکت داخڵکردووە
2. پەیوەندی ئینتەرنێت پێویست نییە - لە ناوخۆی ئامێرەکەت هەڵدەگیرێت
3. ئەگەر کێشەکە مایەوە، ئەپ بکەوە و دووبارە بیکەوە
```

### کێشە: Widget نوێ نابێتەوە

**چارەسەر:**
```
1. کاتێک داتایەک تۆمار دەکەیت، چەند چرکەیەک چاوەڕێ بکە
2. Widget لابدە و دووبارە زیادی بکە
3. ئەگەر کێشەکە مایەوە، ئامێرەکە restart بکە
```
